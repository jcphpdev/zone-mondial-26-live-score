PUBLICATION À DISTANCE

Connexion GitHub

Utilisez un token limité au dépôt avec la permission Contents: Read and write. Il reste uniquement dans cet onglet et disparaît lorsque vous le fermez.

Non connecté

Ne saisissez pas votre mot de passe GitHub. Créez un token fine-grained réservé au dépôt jcphpdev/zone-mondial-26-live-score.

Publier les modifications

Un commit sera créé automatiquement sur GitHub. L’overlay recevra ensuite la nouvelle version dès que GitHub Pages l’aura publiée.
